# #ЩасСпасу

Bilingual React website for the `#ЩасСпасу` initiative.

## Tech stack

- React 18
- React Router
- i18next / react-i18next
- react-slick
- Create React App

## Scope

- 3-route brochure-style frontend
- Ukrainian / English localization
- responsiveness
- accessibility and interaction cleanup
- static deployment support
